## 仓库说明

课程主题聚焦于“硬科技 + 创新创业”的结合，强调从技术可行性、市场需求、商业模式和落地路径四个维度，逐步完成一个具有现实价值的项目构想。

当前仓库以 Markdown 文档为主，便于持续迭代、版本管理和阶段性提交。

## 当前阶段重点

现阶段的主要任务是完成前期选题收敛，明确后续课程项目的主攻方向。  
从目前的整理结果来看，较适合继续深入的方向包括：

* 基于 RAG 的企业销售知识库与智能应答助手
* 基于 AI 语义理解的合同风险扫描与合规辅助系统

这两个方向兼具技术深度、行业应用价值与课程展示可行性，适合进一步展开调研与方案设计。
