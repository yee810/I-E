# 硬科技创新创业课程仓库

本仓库用于整理《硬科技创新创业》课程的阶段性提交物，记录小组对课程的理解、个人选题思考，以及后续项目调研与方案迭代过程。

---

## 仓库说明

课程主题聚焦于“硬科技 + 创新创业”的结合，强调从技术可行性、市场需求、商业模式和落地路径四个维度，逐步完成一个具有现实价值的项目构想。

当前仓库以 Markdown 文档为主，便于持续迭代、版本管理和阶段性提交。

---

## 当前文档

### 1. [提交物 01：课程理解单](./01_Course_Understanding_Group.md)

该文档主要整理了小组对于课程核心逻辑的理解，包括：

* 对“硬科技创新创业”课程定位的认识

* 5 个潜在项目方向的概览与详细说明

### 3. [提交物 03：个人资料卡](./03_Individual_Profile_Cards.md)

* 三位成员的个人能力与协作特点梳理

### 4. [提交物 04：项目初步规划](./04_Project_Preliminary_Plan.md)

* 小组分工方式与当前风险应对思路

### 5. [提交物 05：检索式与调研分工表 V1](./05_Search_Queries_and_Research_Assignment_v1.md)

* 法规、技术、竞品三个维度的检索式设计
* 不少于 10 条的初步资料条目清单
* 小组调研分工表与 3 条待核验问题
